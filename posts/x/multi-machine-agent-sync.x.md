# X Stubs - Multi-Machine Agent Sync

## EN

```text
One pain I keep hitting in AI engineering:

the same useful workflow pieces need to exist across several machines and isolated runtimes.

Copy-paste does not survive that for long.

I turned my current workaround into a small public repo:
https://github.com/AI-Neighbors/skillsOS

It is a thin sync CLI plus one public pack.

If this pain looks familiar:
star it, fork it, or send a PR.
```
